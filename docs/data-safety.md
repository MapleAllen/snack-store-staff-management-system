# 数据安全与备份

## 存储方式

工资工作区保存在本机浏览器存储中，不会由应用主动上传。工作区、自动恢复点和手动 JSON 备份均未加密。Electron 桌面版会在应用数据目录维护有限数量的恢复点。

自动恢复点的触发时机包括每日首次启动、恢复数据前和完成月结后。恢复流程会校验备份类型、存储键和基本工作区结构，并要求用户再次确认。

## 操作建议

- 首次录入真实数据前，确认 Windows 用户账户和磁盘访问权限。
- 每次升级、迁移设备和批量修改前导出 JSON 备份。
- 将手动备份复制到受控的其他介质，不要只保存在同一块磁盘。
- 不要通过公开 Issue、聊天记录或源码仓库传输备份。
- 使用独立 Windows 账户和 BitLocker 降低本机明文数据暴露风险。
- 月结后抽查正式导出与冻结快照；发现错误时记录原因后再解锁。

## 已知边界

本项目没有本地数据加密、云端冗余、多人权限、审计服务器、远程擦除或自动更新。设备损坏、恶意软件、共享 Windows 账户和未加密备份均可能导致数据丢失或泄露。
