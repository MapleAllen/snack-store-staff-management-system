# 安全政策

## 支持范围

仅 `2.x` 最新版本接收安全修复。旧内部安装包不属于公开支持范围。

## 报告安全问题

请使用 GitHub 的 **Private vulnerability reporting** 私下提交，不要创建公开 Issue。报告应包含受影响版本、复现步骤、影响范围和缓解建议，但不得附带真实员工或工资数据。

## 数据与发布边界

- 工作区和备份保存在本机且未加密；应用依赖 Windows 账户、磁盘加密和物理访问控制。
- 项目当前不发布 Windows 可执行文件。任何第三方提供的 EXE 都不是受信任的官方产物。
- 自动恢复点不能替代异地备份，也不能防止设备丢失、恶意软件或共享账户造成的数据泄露。
